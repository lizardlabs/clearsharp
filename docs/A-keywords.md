# Appendix A â€“ Complete Keyword List

This appendix defines the **authoritative list of reserved keywords** in the ClearSharp language.

Keywords are:
- case-insensitive
- reserved (cannot be used as identifiers)
- grouped by conceptual area

Future versions may add keywords, but existing keywords will not change meaning.

---

## 1. Structural Keywords

Used to define program structure and blocks.

- `service`
- `module`
- `action`
- `rules`
- `handler`
- `query`
- `request`
- `message`
- `messagebus`
- `datasource`
- `api`
- `generate`

Block terminators:

- `end`
- `end if`
- `end for`
- `end action`
- `end service`
- `end rules`
- `end handler`
- `end query`
- `end request`
- `end message`
- `end datasource`
- `end api`
- `end generate`
- `end select`
- `end try`
- `end with`
- `end parallel`
